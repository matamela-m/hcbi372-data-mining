# HCBI372 — K-Means Clustering Analysis

**Course:** HCBI372 Data Mining  
**Dataset:** Volunteer engagement data (21 records, 6 skill categories)  
**Method:** K-Means clustering (k = 3) on Age and ShiftsCompleted

---

## What It Does

`analysis.py` performs a full K-Means clustering pipeline on `HCBI_SF2_user_data.csv`:

1. Loads and explores the dataset (shape, dtypes, descriptive stats, missing value check)
2. Standardises `Age` and `ShiftsCompleted` using `StandardScaler`
3. Runs the Elbow Method (k = 1–8) to confirm the optimal cluster count
4. Fits K-Means with k = 3 and assigns cluster labels to each volunteer
5. Prints a cluster profile summary table to the console
6. Generates and saves four PNG charts to `output/`
7. Produces a fully self-contained HTML report — `output/flexifinance_cluster_report.html` — with all charts embedded as base64 images

## How to Run

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Run the analysis

```bash
python analysis.py
```

The script prints live progress to the console as it works through each step. When it finishes, all output files are in the `output/` folder.

### 3. View the report

Open `output/flexifinance_cluster_report.html` in any web browser. The file is fully self-contained — no internet connection required after load (charts are embedded as base64).

## Cluster Summary

| Cluster | Label | Avg Age | Avg Shifts | Avg Happiness |
|---|---|---|---|---|
| 0 | Low Engagement | ~22 | ~6 | ~6.3 |
| 1 | High Engagement | ~65 | ~38 | ~9.6 |
| 2 | Moderate Engagement | ~34 | ~18 | ~7.8 |

The key finding: **sustained participation is the primary driver of volunteer happiness**, not age alone. Volunteers who complete more shifts consistently report higher happiness scores, regardless of skill type.
